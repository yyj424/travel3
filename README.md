# travel3  🛫 
###### _여행 계획 및 컨텐츠 공유 플랫폼_   (21.03.26 updated)


### Introduction 
+ 여행 계획기능 (장소 검색 후 코스에 추가)
+ 여행 사진글 공유 기능
+ 장소에 대한 리뷰기능

+ (밴드 및 카톡 기능) + 여행계획 및 장소 리뷰 기능
